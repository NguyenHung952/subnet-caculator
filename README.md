# 📖 Giới thiệu

**Subnet Calculator** là một web tool giúp tính toán và phân tích thông tin mạng con (Subnet) từ địa chỉ IP và Subnet Mask.

Công cụ hỗ trợ:

- Xác định Network ID
- Tính Broadcast Address
- Tính dải IP hợp lệ
- Xác định số lượng Host khả dụng
- Chuyển đổi CIDR / Subnet Mask
- Phân tích thông tin mạng nhanh chóng
